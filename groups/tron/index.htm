2004-04-20 - TRON之父首次来华
    TRON 是目前世界上运用最广泛的嵌入式操作系统，已广泛应用在汽车电子、手机、数码相机和其它电子产品。在家具、工业产品设计、建筑、城市、博物馆、购物超市、物流网等领域已大量采用基于TRON 标准的嵌入式网络计算机。

被称为TRON之父，在日本IT界声望远胜于比尔。盖茨的国际顶级IT专家——坂村健先生，应《微电脑世界》杂志社之邀将于2004年4月22日首次来华公开演讲，为中国企业及用户展示其在随 时随地（泛在）计算技术领域的最新研究成果。

坂村健先生是国际上极负盛名的IT学者，他在1984年发布的TRON系统目前已经安装到了全球30~40亿台数码相机、汽车引擎控制器等电子产品中，远远超过Windows 1.5亿台PC的装机量。

坂村先生领导提出的Ubiquitous Network Society（泛在网络社会）与比尔。盖茨提出的“无缝计算”（Seamless Coumputing）战略异曲同工，目标都是为了建立未来随时随地计算和随时随地信息处理的IT平台和IT环境。

此次，坂村健先生应邀来京出席于4月22日召开的由中国计算机世界传媒集团主办、《微电脑世界》杂志社承办的2004中国信息技术趋势大会，并为大会作题为“随时随地计算技术的现在和未来”的主题演讲，国内用户可以从中了解随时随地计算技术的最新信息和未来趋势。

另一位在RFID的研究及应用开发领域具有很深造诣的日本IT业界知名专家——山本修一郎先生也将应邀为本次大会作特邀演讲，向国内企业及用户介绍最新的RFID技术和应用状况及未来发展趋势，同时HP、IBM等IT领导厂商也将为国内企业和用户带来各领域最新的IT技术。

2004-03-17 -采用TRON技术的5厘米超小电脑

    日本Personal Media公司采用T-Engine base开发出了一款超小的Teacube computer，而且于去年的TRON SHOW展示了，Teacube 采用NEC Electronics MIPS 核心处理器，VR5701，配备了64MB RAM 和16MB的闪存，尺寸仅为52x52x45mm，输出方面有USBx2, RS-232Cx2, CF Card Slot, 100Bse-TX Port, CRT Connector, eTRON 卡插槽, Mic jack, 和耳机。最大显示分辨率为1280x1024，65,536色。系统是基于T-Kernel 和PMC T-Kernel。
