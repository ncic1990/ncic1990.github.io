R&D Experiences

Memory Box

   Memory box is key part of DSAG architecture and focuses on memory hiararchy problems.
    In today's computer architecture, CPU and memory are tight-coupled. This results in many difficulties. First, in some computer memory is idle, but others are out of memory, and they cannot share; second, memory crash will result in the whole system becomes invalid; third, memory upgrade always encounters compatibility problem, some memory dimm cannot used by other mainboards.
   Memory box depart some memory from CPU and congregate them to a rack or box, to provide memory service, we call it Remote Memory Box(RMB). First, it's highly shared and dynamic, and client host can get as possible as more memory from it; it has high bandwidth communication channel and support scalable client number; It can hide the difference of vary memory modules, providing uniformed access interface; memory capacity is scalable; easy to adapt different kinds of memroy module, without change the architecure.
   It can be used as swap device instead of harddisk; as quick checkpoint device; support process migration application; speed up DSM system.
   While this new architecture brings new challenges. We will give approach to solve this problems.

Reconfigurable Optical Interconnect System

   The Raindrop System, name of our optical interconnet evaluation system, is one of the fundamental parts of DSAG. The Raindrop uses DDR interface as commmunication channel with computer system, and uses FPGA to implement DDR control logic. RocketIO Multi-Gbit tranceivers in Xilinx FPGA are used for high speed transmit and receive logic. 12-way VCSEL electronic-optical and PIN optical-electronic modules are used for optical interconnect. With DDR266 and 12 RocketIO transceivers, the total bandwith can reach 25Gb.
