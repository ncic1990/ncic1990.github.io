 一种新型Internet内容版权保护计算机制

 
一种新型Internet内容版权保护计算机制


我们研究的主题是Internet 内容的版权保护，它是属于Internet 信息安全的范畴，我们可以将网上的信息和数字内容划分为四类：普通信息、信息服务商品、数字作品和绝密信息。Internet 版权保护主要是针对数字作品和信息服务商品两个方面，以版权保护技术为基础的信息商务将是Internet 电子商务的一个大的应用领域。

我们研究的主要目的是设计出一种基于加密的纯软件的解决方案，包括软件系统的体系结构和安全性两个方面的问题。

体系结构在Internet内容保护时在网络上采用怎样的安全体系结构？在Web的客户方采用怎样的安全软件框架？进一步，对于Web浏览器上的可信环境提供什么服务？该软件结构会不会导致其他的安全问题？

安全性对于内容版权保护的含义是什么？在安全内容分发时对于Web客户方的威胁模型是什么？浏览器的可信环境提供什么具体操作保证内容的安全性？对于内容的安全协议是怎样?

本论文分为六个部分，第一部分简单介绍了版权保护问题的历史渊源、哲学背景以及相关法律和技术发展，同时提出了在Internet 发展中版权保护技术的必要性。第二部分我们首先简单介绍了版权保护技术的现状，介绍了基于加密和基于水印两大流派的研究。讨论现有的版权保护与内容保护中的技术问题和不足，并对于相关的研究项目和关键的技术作了一个比较。第三部分我们介绍安全内核的概念。第四部分首先我们回顾了安全技术的加密基础，同时我们给出了信息商务的逻辑商业模型，最后给出可相应的版权保护的基于安全内核的双重认证通道的安全协议。第五部分我们采用逻辑分析方法分析了信息商务中版权保护协议的责任性的属性，我们扩展了Kailar 用于分析电子商务协议的责任性逻辑的语言，借助该语言分析了本协议的机制。第六部分我们给出了系统实现的安全问题的考虑，特别讨论了Netscape Plugin 构成的安全可信环境的安全性，同时我们给出了量化的分析数据以及与其他相关系统和实现方法的比较，说明了我们版权保护协议设计的特色，并且我们也采用了攻击分析方法对于具体系统的攻击逐个进行了分析。最后我们给出了文章的结论以及今后进一步的研究方向。

网上内容的版权保护技术问题的解决，不仅有助于网上的数字作品的发行，实现网上关于数字作品这种商品的电子商务买卖活动的技术基础，而且网络上信息商品的销售和使用，同时网上内容的分发和使用也是数字化图书馆中的一个重要组成，版权保护技术使得用户可以有效地在网上使用数字化图书馆中的内容，对于本问题的研究也有助于数字化媒体内容的发行。特别本问题研究的安全模型也可以为数字视频/音频广播中的内容保护和内容管理的解决方案。

关键词Internet、 版权保护、安全内核、安全协议、责任性逻辑、浏览器、水印、数字作品

My research topic is Internet content copyright protection. Many e-commerce systems sell valuable media content products in the Web. With the development of content e-commerce, electronic payment systems have made much progress, but copyright protection problem has still been unsolved.

Recently, digital watermarking techniques have received great attention to protect intellectual property of digital content, but many fatal attacks, such as IBM attack make the watermark useless at all. While several e-commerce system for content commercialization have appeared, such as IBM's Cryptolope, InterTrust's Digibox , SoftLock of SoftLock Inc, SoftSeal of Breaker Technologies, e-book technologies. etc. the kernel structure of the technology has not been opened. Some of the same key questions have still been unanswered.

Structure: What software structuring principles are appropriate for
Internet content copyright protection? Furthermore, which service
should be provided by secure kernel? Is it necessary to build a secure
kernel?

Security: What is the meaning of security for copyright protection?
What is the thread model of secure content distribution? What
execution guarantees can License Engine provide?

In my Ph.D thesis, my objective is to explore the use of public-key
cryptography to prevent the distribution of illicit electronic content. In Section 1, we introduce the concept of copyright , and discuss the application of copyright protection for e-commerce. In Section 2, we outline the state of art of copyright protection techology, and explain the encryption-based copyright protection technology and watermark-based copyright protection technology. At last, we analysis the weakness of watermark technology. In Section 3, we mainly introduce the concept of secure kernel. In Section 4, we retrospect the encryption technology and secure authenticate protocol, meanwhile, we present the definition of dual- authenticate- channel-protocol model. In Section 5, we use accountability logic to analyze the copyright protection protocol. In Section 6, we outline of thread model and state some
assumptions about the executive environment on the Web Browser(Netscape
Navigator). At last, we give the software architecture of e-commerce
application that runs over License Engine,and discuss a number of attacks and related protection mechanism. Meanwhile, we give the software architecture of e-commerce application that runs over License Engine(secure kernel) in the Web Browser Plugin.

My main contribution is providing a secure and practical solution for the copyright protection. We use secure kernel mechanism plus security protocol in the copyright protection. When we use the Netscape as the copy defensive mechanism to screen the content to hard disk directly, License Engine plays an important role in the copyright protection. So License can be viewed as the electronic copyright. The computing mechanism for the License support different operating system platform and almost all Internet content format. It is a very practical solution for the Internet content copyright protection.

Key Word: Internet, Copyright Protection, Secure Kernel, Security Protocol, Browser, Watermark Digital Work
