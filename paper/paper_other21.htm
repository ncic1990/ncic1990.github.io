 基于机群的并行计算环境的研究、设计与实现

 
基于机群的并行计算环境的研究、设计与实现


我是北京科技大学信息工程学院计算机系计算机应用技术专业硕士研究生张剑峰。我论文的题目是“基于机群的并行计算环境的研究设计与实现”。我作为中科院计算所客座研究生于1999年11月至2001年2月参与了国家“863”计划的重点项目-曙光3000超级并行服务器。该项目被称作是“863”的一面旗帜，现已正式通过国家鉴定。其中通信子系统是整个项目最重要的组成部分，而MPICH-BCL3又是通信子系统的重要组成部分，因此本课题具有重大意义与价值。

机群是一组通过高性能网络互联起来的计算机系统（节点机）组成的具有单一系统映象的高可用、高性能、高可扩展性的计算机集群系统。机群的发展与机群并行计算环境的发展是分不开的。机群的并行计算环境从编程模型上一般分为“基于消息传递的模型”和“基于共享变量的模型”两种，从构成上讲包括资源管理、通信子模块以及协同操作子模块。

本文结合作者MPICH-BCL3（基于曙光3000的MPI）的开发经历，首先介绍了机群的概念，然后比较详细的介绍了曙光3000的软硬件结构及节点配置等，其中还对Myrinet高速网络、曙光3000特有的精简通信协议BCL、曙光机群的资源管理系统RMS做了相关的介绍。随后对并行计算环境的概念及MPI与PVM作了讲述。

论文的重点是详细介绍如何实现MPICH-BCL3，其中涉及到MPICH的介绍、MPICH-BCL3实现的整体构思、MPICH-BCL3代码包的构成、具体流程的分析，还介绍了本课题中具有创新性的完整3way协议的实现、通信长度不匹配的处理、集群通信的优化三部分内容，最后是正确与稳定性测试、性能测试及分析比较。

关键字： 机群、并行计算环境、MPI、MPICH、机群通信

Dawning 3000 super parallel server is one of the most important projects in National "863" project and has identified successfully. Performance of cluster communication system is one of the key factors, which determine the performance of the cluster system. MPICH-BCL3 is one of the most important components in communication system, so it has great value.

This paper is based on the development practices on the Dawning 3000 MPICH-BCL3 project.In this paper,we first give an overview of the background knowledge in the following aspects: clusters, the software and hardware architecture of Dawning 3000, Myrinet high-speed network, BCL3(a simplified communication protocol designed by Dawning 3000 project) , RMS(Resource Management System) , MPI(Message Passing Interface) and PVM(Parallel Virtual Machine).

The emphasis of this paper is the design and the implementation of MPICH-BCL3. We discuss what is MPICH, the global idea of how to implement MPICH-BCL3,the architecture of MPICH-BCL3 code package and the details of program stream. In my project, the implementation of completed 3way protocol, the method to dealing with the communication length and the optimization of collective communication are creations in this project. Finally, the performance evaluation is also provided.

Keywords: clusters, parallel computing environment, MPI, MPICH, cluster communicaiton.
