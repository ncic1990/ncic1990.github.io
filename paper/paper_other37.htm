 虚拟主机系统

 
虚拟主机系统


近年来，信息服务系统在商业领域得到日益广泛的应用，人们对它的性能和效率也提出了更高的要求。ISP不再仅仅满足于在一台物理机器(特别是高性能机)上只给一个公司提供网络站点，而应该可以给多个公司同时提供站点服务，这就需要虚拟主机技术。

本文首先讨论了虚拟主机的定义、关键问题、技术路线以及评价标准，然后综合分析了现有的各种实现方法。在此基础上，我们提出了用于曙光大型信息服务系统的三层结构虚拟主机技术。相比于已有的技术，三层结构虚拟主机技术的突出优点就是有很好的灵活性，可扩展性和兼容性。根据三层结构的虚拟主机技术，本文阐述了WWW虚拟主机系统的工作原理，总体设计和实现方案，并讨论了FTP和EMAIL虚拟主机系统的设计。

文中还讨论了现代网络交通中的自相似特性和其对系统性能的影响。在此基础上设计并实现了一个性能测试平台，进行了WWW虚拟主机系统的性能分析和测试，同时和其他解决方案进行了对比。

最后，我们总结各种虚拟主机技术的优缺点，给出了虚拟主机技术在单一映象的机群系统中的应用，并讨论了存在的问题和进一步的工作。

Recently, information service systems are widely used in the commercial fields. At the same time, there are more and more intensive performance and efficiency requirements on them. ISP doesn't satisfy with installing only one network site of one company on a computer (especially high performance computer) any more. In one computer, there should be many company network sites running simultaneously. Therefore, Virtual Host Technology is needed.

In this thesis, we firstly discuss the definition, key issues and technique strategies of Virtual Host Systems, and analyze all kinds of existing solutions. Then we present the Three-Layer Virtual Host System used in the Dawning Large Scale Information Service System. Three-Layer Virtual Host System's main advantages are flexibility, scalability and compatibility. Based on the Three-Layer Virtual Host System, we design and implement one WWW Virtual Host System and carefully discuss the design of FTP and EMAIL Virtual Host Systems.

In this thesis, we also introduce the self-similar property of modern network traffic and discuss its effects on the system's behavior and performance. Based on these studies, we design and implement a performance test platform for WWW Virtual Host System. The test results of all kinds of solutions are compared.

Finally, this thesis summarizes the characteristics of all kinds of Virtual Host Systems and discusses their limitation and future work.
