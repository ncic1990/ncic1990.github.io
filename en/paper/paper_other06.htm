 分布式文件系统吞吐率优化研究

 
分布式文件系统吞吐率优化研究


近年来随着计算技术的迅猛发展，I/O瓶颈问题日益严重。分布式环境中用户对高性能的文件存取能力的需求使得该问题更加突出。因此提高分布式文件系统的性能具有重要的意义。

吞吐率是评价文件系统性能的一个重要指标，本文对吞吐率优化进行了一些探索工作。本文介绍、分析了NFS和PVFS两个具有代表性的文件系统，对NFS和PVFS的吞吐率进行了详细测试。在详细分析了测试结果的基础上，指出了影响NFS和PVFS吞吐率的主要因素，提出采用多成分的路径名查找来提高分布式文件系统吞吐率，并在PVFS上进行了实现。通过对实现前后的PVFS吞吐率进行测试，论文指出多成分的路径名查找对提高分布式文件系统的吞吐率是十分有效的。

论文的绪论部分首先介绍了分布式文件系统的特征、设计中面临的主要问题和缓存技术，然后介绍了论文的工作背景以及论文的组织。

论文在第二章主要介绍了NFS的特性、结构、控制流程、协议和具体实现，论文的第三章介绍了PVFS的结构、控制流程和主要文件操作流程。

论文的第四章提出了分布式文件系统性能开销分析的一种时间开销分类方法，对NFS、PVFS进行了吞吐率测试和跟踪，并对得到的结果进行了统计和分析。根据分析的结果，指出了影响分布式文件系统吞吐率的关键因素，并指出了PVFS在结构、设计上的不足之处。

第五章根据第四章中得到的结论，提出采用多成分的路径名查找来优化分布式文件系统的吞吐率，并且将该方法在PVFS上做了实现。第五章还给出了优化前后的PVFS吞吐率测试数据，并根据该数据指出多成分的路径名查找对于优化分布式文件系统的吞吐率是十分有效的。

最后，第六章给出了本论文的结论，并讨论了进一步研究的方向。

关键词 分布式文件系统、吞吐率、可扩展性

THROUGHPUT IMPROVEMENT OF DISTRIBUTED FILE SYSTEMS
Yuzhuang　Hu（Computer Architecture）
Directed by Professor Dan Meng

With the rapid developments of computing technology，the problem of I/O bottleneck becomes more and more severe. It is aggravated because of the demand of high performance file accessing ability in the distributed environment. Improving the performance of distributed file systems is crucial to the whole systems.

Throughput is an important metric of the systems' performance. This paper works on the issue of throughput improvement of cluster file systems.First，this paper introduces two classical file systems-NFS and PVFS.Then，based on comparing and analyzing the throughput of these two systems，this paper points out the key factor which influences the throughput mostly.Finally，the author presents his idea of multi-component lookup which is used to improve the throughput of cluster file systems. This method proves very useful according to the testing results before and after this implementation.

In Chapter 1，this paper introduces the general characteristics、several design issues and caching technique of cluster file systems.Then the background for this paper is discussed.

Chapter 2 introduces the structure、control flow、protocol and implementation of NFS.Chapter 3 focuses on the structure、control flow and implementation of PVFS.

In chapter 4，after presenting his method of classification of time spent in distributed file systems，the author turns to the testing and the analysis of the throughput of NFS and PVFS.In this chapter，the key factor which influences the throughput mostly is discussed.Also in this chapter，the author points out the shortcomings of the design and implementation of PVFS.

According to the conclusions of Chapter 4，the author describes a design and implementation of multi-component lookup，which is used to improve the throughput of distributed file systems. The performance before and after this implementation has also been compared.
This paper ends with the conclusions and future research directions which are given by the author.

Key words Distributed file systems，Throughput，Scalability
