 机群服务器聚集系统研究

 
机群服务器聚集系统研究


服务器聚集模式是90年代后期出现一种的既保持了客户机/服务器模式的开放性、可扩展性优点,又具有传统的终端/主机模式的集中、易于管理等特点的信息系统模式。机群系统是实现可扩展大型服务器聚集系统的理想平台。本文主要研究利用机群系统实现可扩展大型Internet服务器聚集系统的关键技术，并着重从单一登录点技术、高可用管理技术和可扩展并行Internet应用三个方面进行了研究。
本文首先全面分析了实现机群服务器聚集系统中的各种关键技术及其发展现状。并基于M/G/1 PS模型分析了机群服务器性能，给出并证明了在随机分配策略下异构机群达到最小平均等待时间的条件。

随后本文重点讨论了机群服务器聚集系统的核心技术--单一登录点技术。在详细分析了5种基于传输层包转发的单一登录点技术之后，我们给出了在曙光2000超级服务器上的单一登录点软件 PowerRouter的设计实现和性能分析。PowerRouter实现了直接包转发技术，具有不修改源数据包、不修改节点操作系统、直接返回数据等优点。实验表明PowerRouter能够很好的支持大型Web等可扩展Internet应用。本文进一步讨论了通过应用层转发实现单一登录点的优缺点，并提出了一种结合传输层快速转发和应用层灵活处理的优点的单一登录点技术--延迟选择网络地址翻译技术。

一个集中、高效的系统管理软件是服务器聚集系统必不可少的部分。本文设计并实现了曙光服务器聚集应用管理系统DSC。DSC在设计中实现了与操作系统、应用软件和机群节点的独立性。其虚拟服务和负载平衡设计将网络并行应用的管理和高可用性管理结合在了一起，并具有全局资源的自动分配和回收，灵活的故障事件处理，可靠通信机制，动态配置修改等功能。整个系统结构清晰，功能完整，配置灵活，是一个有效的管理工具。

最后本文设计并实现了一种具体的服务器聚集应用--可扩展大型e-mail服务器。根据e-mail应用用户之间具有独立性的特点，并结合前两部分的研究结果，本文提出的一种层次型分布式大型e-mail系统结构，突破了传统集中式存储系统在扩展中遇到的限制。通过分布式存储解决了存储容量扩展的瓶颈问题，通过基于三级单一登录点的负载平衡和功能单元并行化解决了系统性能扩展的瓶颈问题。系统中各节点功能划分清晰，便于采用不同的优化配置。本文给出了在曙光2000上的FreeMail的系统实现和性能分析。实验表明这种结构具有良好的性能和可扩展性。

【关键词】 服务器聚集，机群，单一登录点，负载平衡，可扩展性、可用性、e-mail, 大型系统，分布式存储

Research on Cluster Server Consolidation System
Mingyu CHEN (Computer Architecture)
Directed by Professor Guojie LI

Server Consolidation is a new IT model that emerged in later 1990's. It combines the openness and scalability of distributed client/server model and the centralization and easy-to-management features of mainstream model. Cluster system is the best platform to implement scalable large-scale server consolidation system. In this dissertation, key technologies to build scalable large-scale Internet server consolidation system are studied, with the emphasis on single entry point (SEP), high availability management and scalable parallel Internet application.

After analyzing all aspects of designing cluster server consolidation system, we study the performance model of network Server based on M/G/1 PS queue. The condition for minimum average-wait-time under random dispatch algorithm is presented and proved.

Then this dissertation concentrate on the implementation of SEP. Five technologies based on packet forwarding are discussed in detail. Then the design, implementation and performance analysis of the SEP software，PowerRouter on the Dawning 2000 Cluster，are given.

PowerRouter adopted the direct forwarding technology. It has the advantages of direct reply without source package modification and OS modification. Benchmark shows that PowerRouter can well support large Internet application such as large-scale web server. The technology based on application layer forwarding is also discussed. Then a new technology, delayed-decision network address translation, is presented, which combined the efficiency of transport-layer forwarding and the flexibility of application-layer forwarding.

A central, flexible system management software is necessary for each server consolidation system. In this dissertation, the design and implementation of application-management software for Dawning Server Consolidation (DSC) are presented. DSC is independent of OS, application software and special nodes. The virtual service and load balancing design combined the parallel-application management and high availability control. It also has mechanisms for global resource allocation and reclamation, flexible event processing, dependable communication, dynamic configuration etc. The system is a clear and flexible management tool.

Finally, a marketable server consolidation system - a scalable large-scale e-mail server is introduced. Based on the characteristic of independence among e-mail users, along with the studies above, a hierarchical large-scale e-mail server architecture based on distributed-storage is presented. It solves the problem faced by traditional central-storage mail servers when scaling up. The distributed storage design avoids capacity bottleneck. The load balance mechanism based on three levels SEP and the parallelization of each type of functional unit eliminate the performance bottleneck. The nodes have clear functional partition for special configuration optimization. The implementation and the performance analysis of FreeMail on Dawning 2000 system are given. Tests showed good performance and scalability in test.

【keywords】 server consolidation, cluster, single entry point, load balance, scalability, availability, e-mail, large-scale system, distributed storage
