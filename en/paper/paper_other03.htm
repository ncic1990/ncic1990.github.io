 高效并行计算系统中的计算模型与计算网络

 
高效并行计算系统中的计算模型与计算网络


提高并行计算机系统的通信效率是并行处理的关键课题，本文围绕着通信效率，选择了计算模型和互连网络作为研究方向。针对并行程序中的通信效率问题，分析了产生通信拥挤的三个因素，即BSP程序引起的通信拥挤、网络拓扑结构引起的通信拥挤和网络嵌入算法不当引起的通信拥挤。本文的主要创新点如下：

①提出了一类基于Petersen图的互连网络RP(k)。该互连网络具有短的网络直径、简单的拓扑结构、方便的路由策略；该网络硬件复杂度低、实现简单。本文对该网络进行了扩展，提出了RP(P,k1,k2)网络拓扑，并讨论了其性质。

②设计了RP(k)和RP(P,k1,k2)网络上的路由算法，主要设计了点点路由、置换路由、广播路由和All-to-all路由算法。这些算法具有较高的通信效率。

③提出了将Ring和Mesh 嵌入RP(k)网络的算法, 证明了RP（k）网络是一个Hamiltonian 图，讨论了RP(k)网络的容错问题。同时，也设计了将2-D mesh 嵌入RP（k）网络的方法，取得了良好的嵌入性能。

④基于光RP(k)网络，设计了一个实现n维Hypercube 通信模式的算法和一个将n维Hypercube 嵌入环网络的算法，后者比已有算法有较大的改进。

⑤提出了一种存储效率分析方法及异步CSA-BSP计算模型。

⑥提出了网络最优分划的概念，给出了一种评价互连网络性能的方法。

How to enhance the communication efficiency is regarded as one of the key topics in parallel processing. Here the parallel computation models and interconnection networks are chosen as the key topics of our researches. The main achievements of this thesis can be summarized as follows:

①Based on the Petersen graph, the RP(k) network is proposed. The RP(k) has many properties, like short diameter, simple topology, flexible routing strategies. To enlarge the size of this network, the RP(P,k1,k2) network is proposed.

②Based on the RP(k) and RP(P,k1,k2) networks, the routing algorithms have been designed, which include point-to-point routing, One-to-all routing, permutation routing, and All-to-all routing.

③The embedding algorithms of Ring and Mesh into RP(k) network are proposed. It is proved that the RP(k) is a Hamiltonian graph and has strong fault-tolerant properties.

④ Based on the BSP model, the CSA-BSP model has been proposed. Theoretic analyses and experiment results prove that the CSA-BSP model can guide programmers to write more efficient programs.

⑤Based on the optical RP(k), an algorithm to implement the Hypercube communication is designed. An algorithm to embed the Hypercube into the ring network is also proposed, which improves existing result greatly.
