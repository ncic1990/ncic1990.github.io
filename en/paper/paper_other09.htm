 高效并行计算系统中的计算模型与计算网络

 
高效并行计算系统中的计算模型与计算网络


随着超级计算机体系结构的不断演变，机群逐渐成为并行处理研究的热点和主题。机群系统是指将一组节点计算机通过高性能互连网络连接起来，并作为一个单一系统向其用户提供高性能、高可用服务的系统。机群中的每个节点计算机上安装有完整的商业化操作系统，要使这些系统成为一个单一的系统，对用户提供高性能、高可用服务，就必须使用单一控制点技术（单一系统映像SSI技术之一）。单一控制点是指系统管理员可以从一个单点对整个机群及其各个单独的节点进行配置、监视、测试和控制。系统管理软件是机群系统管理员管理和监控整个机群状态的有力工具，是单一控制点的一部分。由此可见机群系统管理软件在机群系统中的重要地位。

本文作者通过参与国家863计划之曙光3000机群系统管理软件CSMS的研究与开发工作，提出机群系统管理软件中的三个关键技术：异构机群管理，机群的远程管理和机群的全局配置管理。在对这三个关键技术分别进行了实验、分析和研究之后，作者给出了结论。最后，论文向读者展示了曙光3000机群管理系统的部分图形用户界面，并提出了机群系统管理软件今后的发展方向和未来的实现目标。

关键词：机群 单一控制点 系统管理 异构机群 远程管理 全局配置 远程过程调用 网络信息系统 轻量级目录访问协议

With the increasing development of the architecture of supercomputers, cluster has become the hot topic of the parallel computing research. Cluster system, a system that is connected with a group of node computers via high performance network, provides high performance, high availability services to its users as a single system. There is an integrated commercial operating system running on every node computer of a cluster. In order to make the system as a single system providing high performance, high availability services to its users, we must use Single Control Point technology, which is a part of Single System Imager technology (SSI). Single Control Point means that the administrator could configure, monitor, test and control the whole cluster and each node computer in it from a single console. The system management software, as a part of Single Control Point, is a powerful tool used to manage and monitor the status of the whole cluster. So the cluster system management software plays an important role in a cluster system.

Through participating in the research and development of Dawning 3000 cluster system management software (CSMS) of the Chinese National High-Tech Program (863), the author of this thesis provides three key technologies of the cluster system management software: the management of a heterogeneous cluster, the remote management of a cluster and the global configuration management of a cluster. After some elaborate experiments were done and the results of the experiments were analyzed and researched carefully, three conclusions were drawn respectively. At last, the thesis shows some GUIs of Dawning 3000 and the future goal of the cluster system management software to the readers.

Key Words: cluster, Single Control Point, system management, heterogeneous cluster, remote management, global configuration, RMI, NIS, LDAP
